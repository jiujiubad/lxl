网络常识菜鸟笔记 (5) 给自己做一个USB启动盘

by xiaolai on 2009/10/24 · 14 comments

in 网络常识菜鸟笔记

菜鸟的特点就是不停地试用各种软件，安装之后再卸载，也许还要重新安装再重新卸载……这些事儿做多了，系统就乱了、电脑变慢了，系统崩溃了……不测都可能随时发生。所以，菜鸟需要一个启动盘。今天的计算机基本上都支持USB设备启动（并且很多笔记本电脑已经不再内置光驱、软驱了），下面我们就看看如何制作一个USB启动盘。

准备工作：

  1. 你需要准备一个容量至少为128M的U盘（容量超过200G的移动硬盘也可以）——记得先将里面的文件备份好；
  2. 你需要在系统上安装WinRAR软件（或者任何其他可以提取iso文件内容的应用软件，比如UltraISO什么的）；
  3. 下载Active Boot Suite的iso镜像文件：http://ifile.it/jwv50sb (122M)。

在Windows Vista/7 之下将U盘（或USB移动硬盘）连好，而后打开一个命令行窗口（“开始”->“所有程序”->“附件”->“命令提示符”），输入命令“diskpart”之后按回车键。

DISKPART> list disk

磁盘 ### 状态 大小 可用 Dyn Gpt
——– ————- ——- ——- — —
磁盘 0 联机 298 GB 1024 KB
磁盘 1 联机 3830 MB 0 B

DISKPART> select disk 1

磁盘 1 现在是所选磁盘。

DISKPART> clean

DiskPart 成功地清除了磁盘。

DISKPART> create partition primary

DiskPart 成功地创建了指定分区。

DISKPART> select partition 1

分区 1 现在是所选分区。

DISKPART> active

DiskPart 将当前分区标为活动。

DISKPART> format fs=ntfs quick

100 百分比已完成

DiskPart 成功格式化该卷。

DISKPART> exit

退出 DiskPart…

（以上是命令行操作过程，其中文字加重的部分是输入的命令，其他部分是计算机的反馈内容。我的U盘大小是4G，计算机为其标识的大小为3830 MB。）

现在再用WinRAR把Active-BootDisk.iso文件中的所有内容解压缩至移动存储设备的根目录中。

重新启动计算机，在BIOS中将启动顺序设置为USB存储设备优先。

（大多数台式计算机在启动的时候按“DEL”键即可进入BIOS设置界面；大多数笔记本电脑在启动时按F2（或F11，或F12）可以进入BIOS设置界面或者进入“选择启动设备界面”。）

用这个启动盘启动计算机，启动之后进入的是Windows XP的PE系统，运用Active Boot中预装的程序可以备份分区，恢复文件等等，具体其它用处则有待你自己开发了。

另，我把自己的C盘备份完了之后发现其大小为8G左右，于是，就把另外一个16G的U盘做成了启动盘，而后把C盘备份存了进去……是跟Ghost这个软件说白白的时候了。
