为Prezi Desktop设置中文字体

by xiaolai on 2011/12/23 · 35 comments

in 计算机相关

Prezi是个很酷的演讲辅助工具，演示效果比PowerPoint、Keynote炫多了，不仅如此，更重要的是，它在文字、数据视觉化方面确实比其他工具高出了好几个层次。

真正吸引我的，其实不是它那种“炫”的效果——因为某一种效果，看多了就没意思了。最吸引我的是它的信息展示方式——用Prezi很容易做到让观众一直保持关注“the BIG picture”——即，不会像看PPT/Keynote那样，很容易迷失到某个细节里面……

然而，Prezi“很贵”，Pro版本要每年159美元——不是一次性的159美元……

Prezi的缺陷是对中文字体支持较差（尽管也内置了中文字体）——很多中文字符无法显示。Google了一下，发现Prezi Desktop上还是可以加载自定义字体的（网页版不行，Desktop编辑，就只能用159美元的Pro版本）。

点击这里下载字体文件（字体制作者网站）（前两个链接都失效了。昨天[20120718]在Twitter上，@goldengraple告诉我一个新的下载地址：http://vdisk.weibo.com/s/8L12d）：

将解压缩后的文件中所有swf文件拷贝到：

Mac OS X:

~/Library/Preferences/com.prezi.PreziDesktop/Local Store/fonts

Windows 7:

C:\Users\[username]\AppData\Roaming\com.prezi.PreziDesktop\Local Store\fonts

Linux:

~/.appdata/com.prezi.PreziDesktop/Local Store/fonts

注意，要覆盖原有的文件！

创建一个新的Prezi文档。

选择合适的模板。

按下Ctrl+Shift+C打开CSS模板编辑器。

修改@font-face中的src: url(‘xxx.swf’)为GBC Fonts中提供的中文字体，例如：

@font-face
{
src: url(‘STXINWEI.swf’);
fontFamily: body;
}

@font-face
{
src: url(‘STCAIYUN.swf’);
fontFamily: strong;
}

@font-face
{
src: url(‘STXINGKA.swf’);
fontFamily: head;
}

而后点击“Apply”按钮。

另外，iPhone上的Remote Pro可以用来当“遥控器”。
