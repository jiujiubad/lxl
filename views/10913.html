如何在HTML中显示Apple OS X的特殊键盘字符

by xiaolai on 2010/03/07 · 10 comments

in 小技巧

在HTML中，可以用unicode显示Apple OS X上的特殊键盘字符:

⎋ Escape U+238B
⇥ Tab forward U+21E5
⇤ Tab back U+21E4
⇪ Capslock U+21EA
⇧ Shift U+21E7
⌃ Control U+2303
⌥ Option (Alt, Alternative) U+2325
 Apple symbol U+F8FF
⌘ Command (Open Apple) U+2318
␣ Space U+2423
↩ Return U+21A9
⌫ Delete back U+232B
⌦ Delete forward U+2326
﹖⃝ Help U+003F & U+20DD
⇱ Home U+21F1
⇲ End U+21F2
⇞ Pageup U+21DE
⇟ Pagedown U+21DF
↑ Up arrow U+2191
↓ Down arrow U+2193
← Left arrow U+2190
→ Right arrow U+2192
⌧ Clear U+2327
⇭ Numberlock U+21ED
⌤ Enter U+2324
⏏ Eject U+23CF
⌽ Power U+233D
